Quickstart
----------

In a terminal, run the following:

.. code:: bash

   make
   # once the docker stack is running, click the URL starting with
   # http://127.0.0.1:8888/?token=... to access JupyterLab
   #
   # mlflow UI is at http://localhost:5000/
